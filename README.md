## 테이블(표)

position 속성

값 | 의미 | 기본값
--|--|--|
static | 기준 없음 | O
relative | 요소 자신 | X
absolute | 위치 상 부모요소 | X
fixed | 뷰포트 | X



가운데 정렬은 헤드 구분선의 하이픈 양쪽에 콜론: 기호 사용
오른쪽 정렬은  헤드 구분선의 하이픈 오른쪽에 콜론: 기호 사용

값 | 의미 | 기본값
--|:--:|--:|
static | 기준 없음 | O
relative | 요소 자신 | X
absolute | 위치 상 부모요소 | X
fixed | 뷰포트 | X


의미 열은 가운데정렬, 기본값 열은 오른쪽정렬 됨
